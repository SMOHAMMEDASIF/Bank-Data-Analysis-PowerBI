## Overview
This project presents an interactive **Power BI dashboard** built to analyze bank customer and loan data.  
The dashboard helps identify customer behavior, loan performance, and key financial insights to support data-driven decision-making.

## Objective
- Analyze customer demographics and account details
- Evaluate loan distribution and repayment trends
- Identify high-value and high-risk customers
- Provide clear visual insights for business users

## Tools & Technologies
- Power BI Desktop
- Microsoft Excel
- DAX
- Data Cleaning & Transformation

## Dashboard Pages
- Customer Overview
- Loan Analysis
- Account Balance Insights
- Risk & Performance Analysis

## Dataset
- Source file: **Bank data.xlsx**
- Contains customer, account, and loan-related information used for analysis

## Key Features
- Multi-page interactive Power BI dashboard
- KPI cards for quick performance tracking
- Filters and slicers for dynamic analysis
- DAX measures for calculated insights

## How to Use
1. Download the **Bank Data Analysis.pbix** file
2. Open it using **Power BI Desktop**
3. Load or refresh the dataset if required
4. Navigate through pages using the tabs

## Project Highlights
- Improved visibility into customer and loan metrics
- Enabled trend analysis across multiple financial dimensions
- Designed for clarity, usability, and business understanding
